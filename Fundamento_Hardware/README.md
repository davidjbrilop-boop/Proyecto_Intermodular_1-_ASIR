# Objetivo


**Un cliente propietaria de una licencia de venta de tabacos y timbres va a abrir una nueva linea de negocio en su estanco en el que nos indica que quiere ofertar el alquiler de maquinas de tabaco para la venta de cajetillas de tabaco en segundo canal (bares, gasolineras) y poder gestionar tanto el alquiler de las máquinas, como el suministro de producto suministrado.**


**Para la parte de hardware nos indica que necesita dos equipos para el despacho en tienda o mediante llamada telefónica mediante otro equipo en administración (Recepción de pedidos y alta de usuarios) otro equipo en el almacén para la preparación de pedidos, un equipo mas en soporte técnico.**


## 1. Análisis de necesidades de hardware


**La empresa necesita 2 ordenadores por departamento distribuidos en**



| **Departamento** | Equipo |
| :-: | :-: |

| **Administración** | 1 Pc |
| **Almacén** | 1 Pc |
| **Soporte técnico** | 1 Pc |

**Que tipo de equipos se necesitan**


**Recepción**

**	Estos equipos seria suficiente con que fueran barebones o mini ordenadores, así conseguimos que no ocupen un sitio en nuestra recepción que moleste o estorbe al tratar con los clientes.**


**Administración**

**	Este equipo también usaría del tamaño de mini pc puesto que así nos despeja la zona de trabajo mas necesaria para la documentación de empresa.**


**Almacén**

**	Este equipo un Pc de tamaño torre para gestionar el mismo y con la necesidad de un scaner de mano para el tema de trazabilidad**



**Soporte técnico**

**Los PC´s de este departamento no es necesario que dispongan mas de 8GB de memoria RAM.**


**Aula de reuniones**

**	Este departamento instalaremos una torre como el del almacén, para hacer video conferencias o presentaciones que realizan en el negocio para el lanzamiento de nuevos productos.**

### 2. Componentes de un equipo informático


**CPU**


**El cerebro del equipo, interpreta y ejecuta instrucciones, realiza cálculos y procesa datos.**


**Incluye:**

**·La unidad de control (UC): Dirige y controla los datos**

**·La unidad aritmética/lógica (ALU): realiza operaciones matemáticas **

**·Los registros: memoria de trabajo inmediata**


**Placa base**


**Sirve como soporte donde se conectan todos los elementos del sistema entre si**

**Dispone:**

**·Conexiones físicas: para conectar los elementos**

**·Buses: los caminos por donde transcurre la información**

**·BIOS/UEFI: Chip con software necesario para arrancar el equipo y reconocer el hardware instalado**


**Memoria RAM**


**Memoria principal a corto plazo de una computadora. Almacena temporalmente los datos y programas. Mientras reciba energía guarda la información, si tiene un corte de energía esta memoria al ser volátil se pierde**

**Se clasifican:**


|  |  |
| - | - |
| **Tamaño** | **DIMM** |
|  | **SO\_DIMM** |
| **Tecnología** | **DDR** |
|  | **DDR3** |
|  | **DDR4** |
|  | **DDR5** |
| **Según su función especial** | **SRAM** |
|  | **DRAM** |
|  | **VRAM** |






**Disco de almacenamiento**


**Es el hardware diseñado para guardar todos los archivos, programas y el sistema operativo de manera permanente**

**Puede ser:**

**·SSD (como un USB)**

**·HDD (usa discos magnéticos)**


**Fuente de alimentación**


**Convierte la corriente eléctrica de la red en corriente continua.**

**Da energía a cada parte del equipo, estabiliza y protege de picos de tensión o fluctuaciones.**

**Dispone:**

**	· Un conector principal (24 pines) **

**	· Un conector de alimentación procesador (4 u 8 Pines)**

**Esto es lo mínimo, luego depende de los dispositivos que tenga el ordenador tendrá que tener por ejemplo un conector para la gráfica (6+2 pines el más común)**


**Tarjetas de expansión**


**Son componentes que se conectan a las ranuras de la placa base (slots) para añadir nuevas funciones o mejorar las que ya tiene**


**Pueden ser:**

**	·Tarjetas Gráfica (GPU)**

**	·Tarjetas de Red (wi-fi y Ethernet)**

**	·Tarjetas de almacenamiento**

**	·Tarjetas de Puertos (USB/Tunderbolt)**

**	·Tarjetas de Sonido**

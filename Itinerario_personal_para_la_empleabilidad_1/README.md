## 4. Presentación profesional del proyectos

Este proyecto es un encargo que nos realiza y cliente que tiene un negocio de venta de tabacos y quieren abrir una nueva linea de negocio.

Se trata de alquiler de maquinas de venta de tabaco para que sus clientes puedan, disponer de maquinas nuevas sin tener que desembolsar el costo de una maquina nueva.

El estanco quiere llevar un control de las maquinas, a quien se las alquilan, que modelos son, que facturas se hacen para recargar el stock de las mismas.

Con esta linea de negocio que quiere implementar nuestro cliente da servicio a aquellos negocios que quieren ahorrar los costes de tener que adquirir una maquina por ellos mismos a un precio reducido mensual.

Para ello se ha creado una base de datos relacional que incluye todos los datos relativos a tanto de las maquinas, los clientes, proveedores y personal del estanco.

He aprendido que no es solo decir voy a abrir una nueva linea de negocio, sino que lleva una serie de trabajos para desarrollar la idea y que sea eficiente y siempre tiene que ser posible modificar o ampliar las funciones según se vayan necesitando.

## 🚬 Proyecto
El proyecto se trata de un encargo que nos realiza un cliente, que es propietario de una licencia de expendeduría de tabaco, en el que quiere implementar una nueva línea de negocio.

Su idea es procurar un servicio de **alquiler de máquinas de tabaco**, para que sus clientes puedan vender también tabaco en sus establecimientos.

Lo que nos pide es un control de las máquinas que tiene alquiladas, con su duración. Además, también quiere un control de las marcas que venden a cada una de las máquinas y, por ende, a sus clientes.

En este proyecto está desarrollado en cada una de las asignaturas que a continuación detallo:
